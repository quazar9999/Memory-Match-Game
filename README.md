Repeat this for the second coordinate and **don't seperate with spaces**.
Use exactly 4 chars (a reprompt will occur if any of these specifications aren't met).

examples:
1. a1b1->correct
2. A1B1->correct
3. a1 b1->incorrect
4. 1b1a->incorrect
